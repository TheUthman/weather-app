# Minimal theme video sources

The app loads one lightweight video at a time based on the current weather.

| File | Weather use | Source |
| --- | --- | --- |
| `rain.mp4` | Rain, drizzle, and thunderstorm base | User-provided `rainy.mp4` |
| `clear.mp4` | Clear and sunny | [The mountains — Coverr](https://coverr.co/videos/the-mountains-ncfmij5yhj) |
| `clouds.mp4` | Cloudy and overcast | [Cloudy sky — Coverr](https://coverr.co/videos/cloudy-sky-2v00ohjpqe) |
| `snow.mp4` | Snow, sleet, ice, and blizzard | [Snow in a winter wonderland — Coverr](https://coverr.co/videos/snow-in-a-winter-wonderland-ska8xmo3rd) |
| `fog.mp4` | Fog, mist, haze, and smoke | [Foggy forest — Coverr](https://coverr.co/videos/foggy-forest-06yshrbpk8) |

Coverr assets are used under the [Coverr license](https://coverr.co/license).
